# User Help

## What this tool is
Suggest for Purchase (SFP) lets you submit a request for an item the library may consider purchasing.

## Patron submission steps
- **Step 1 — Your info**: Enter your barcode, name, phone, and (optional) email.
- **Step 2 — Item details**: Choose a material type and audience, then enter title/author and (if known) a publish/release date.
- **Step 3 — Confirm**:
  - If the item is found in the catalog, you’ll be asked to place a hold instead of submitting a request.
  - If a close match is found, you can confirm it or continue.
- **Step 4 — Confirmation**: You’ll see the submission confirmation message.

## Common messages you might see
- **“Already in the catalog”**: The item is already owned/ordered — place a hold in the catalog.
- **“Duplicate request”**: Someone has already requested it — check the catalog regularly.
- **“Popular author / auto-order”** (future releases only): For some authors, the library automatically orders new releases. Check the catalog closer to the release date to place a hold.

## Staff interface (for selectors/admins)
- **Requests**: Review submissions and update statuses.
- **Patrons**: Review patron records and Polaris lookup results.
- **Titles**: View and merge deduplicated materials.
